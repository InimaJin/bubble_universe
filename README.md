# The Bubble Universe

I find this effect truly intriguing, so I tried rewriting [this](https://oldbytes.space/@zxdunny/109342959566427298)
piece of code myself!

Initially I had written it in pure JavaScript, which was nice, but later I found that using Rust/WASM for the core calculations made the animation much smoother, especially for larger numbers of iterations (N).

I think that's great.
